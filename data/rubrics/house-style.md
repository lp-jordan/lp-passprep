# Pass Prep House Style Rubric (V1)

## Titles
- Use clear, plain-language titles in Title Case.
- Aim for 6–10 words when possible.
- Avoid hype, slang, and clickbait phrasing.
- Avoid punctuation unless it improves clarity (e.g., a colon for structure).
- Prefer practical nouns/verbs (e.g., “Budgeting,” “Understanding,” “Planning,” “Protecting”).

## Descriptions
- 1–2 sentences.
- Target 18–35 words.
- Start with the “why it matters” or the risk/opportunity.
- Keep tone steady and practical (no motivational language).
- Avoid: “In this lesson, you will…” and other course-template filler.
- Avoid vague promises (“unlock,” “master,” “transform”) unless required by client tone.

## Category Names
- 2–4 words, plain language.
- Should describe the primary theme of the group.
- Avoid cleverness or marketing language.
- Categories should feel like a clean table of contents.

## Voice and Consistency
- Use consistent framing across lessons (risk/benefit, clarity, protection, planning).
- Keep language accessible; define jargon implicitly through context.
- Prefer active phrasing and concrete outcomes.
- Don’t repeat the same sentence openings across many lessons.

## Content Boundaries
- Descriptions should not include legal/tax advice beyond general guidance framing.
- If a topic is sensitive (taxes, contracts), emphasize caution, planning, and verification.

## Quality Bar
- Each title/description pair should be readable standalone.
- Each description should match the title directly (no drift).
- If the lesson is “protective,” make the risk explicit.
- If the lesson is “planning,” make the benefit explicit.
